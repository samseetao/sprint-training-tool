About Sprint Training Tool

Overview

Designed for sprinters and coaches, this tool provides precise timing, distance tracking, and start commands to enhance training performance.

Main Features

  • Timer: Millisecond-precision timer for sprints.
  • Distance Tracking: GPS-based tracking with optional voice announcements.
  • Start Commands: "On your mark → Set → Go" with adjustable/randomized delays.
  • Finish Line Alerts: Notifications when target distance is reached.
  • Session Records: Save time, distance, and notes locally for review.

Privacy & Data

Location data is used solely to calculate distance. All session records and settings are stored locally. No accounts or personal data are sent to the operator.
